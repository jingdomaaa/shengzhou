# shengzhou

### 项目简介

本项目目的是在微信小程序完成员工管理及班组管理,目前完成的有:

1.主界面中各页面间的跳转

2.在心情墙中4种问卷调查

3.个人统计的图表显示

使用技术主要有tarojs,taro-ui,TaroEcharts(完成图表开发)

***注:本项目仅供个人学习使用,未经允许请勿转发及下载***
